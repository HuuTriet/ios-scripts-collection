## ⚙️ Hướng dẫn cài đặt (Shadowrocket)

Để sử dụng module này trên ứng dụng **Shadowrocket**, bạn làm theo các bước sau:

**Bước 1: Thêm Module**
1. Mở ứng dụng Shadowrocket.
2. Chuyển sang tab **Config** (hoặc Modules/Phân hệ).
3. Bấm vào dấu `+` ở góc trên bên phải.
4. Copy và dán đường link Raw dưới đây vào:
   ```text
   [https://raw.githubusercontent.com/HuuTriet/ios-scripts-collection/main/LocketTriet.sgmodule](https://raw.githubusercontent.com/HuuTriet/ios-scripts-collection/main/LocketTriet.sgmodule)
   Bấm Save và gạt công tắc để bật Module Locket Gold by Triet.

Bước 2: Bật MITM (Quan trọng)
Để script có thể can thiệp vào dữ liệu, bạn cần bật tính năng đánh chặn:

Trong Shadowrocket, vào Home -> Cài đặt HTTPS Decryption (hoặc HTTPS Rewrite).

Bật công tắc HTTPS Decryption.

Bấm vào Generate a new CA certificate -> Install CA certificate.

Mở Cài đặt (Settings) của iPhone -> Đã tải về hồ sơ (Profile Downloaded) -> Cài đặt chứng chỉ Shadowrocket.

Vào Cài đặt chung (General) -> Giới thiệu (About) -> Cài đặt tin cậy chứng chỉ (Certificate Trust Settings) -> Bật xanh cho chứng chỉ Shadowrocket.
